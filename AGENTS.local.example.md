# Pixelgrade Assistant Local Agent Overlay

Use this file as a template for private, machine-specific agent context.
Copy it to `AGENTS.local.md` or hydrate it with `bin/bootstrap-private`.

Keep secrets out of committed files.

Suggested sections:
- Local WordPress site URLs and admin testing notes.
- Private issue links, release notes, and operational runbooks.
- SSH aliases or deployment hosts that are safe to mention locally.
- Temporary debugging notes that are not useful to every future clone.
